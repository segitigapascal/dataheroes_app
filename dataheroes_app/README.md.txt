# Data Heroes - LKM Statistik Digital

Aplikasi ini dibuat untuk mendukung pembelajaran Statistik kelas 7 berbasis PjBL, terdiri dari 3 pertemuan:

1. Pengumpulan Data
2. Pengolahan Data
3. Penyajian Data

Aplikasi ini dibuat menggunakan Streamlit multipage dan dilengkapi fitur:
- LKM per kelompok
- Kamus Klasifikasi Jenis Data
- Rekomendasi Grafik berdasarkan jenis data
- Export PDF untuk setiap kelompok
